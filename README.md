# GoldTrace 🪙

A real-time gold and silver price tracker built with React.

## Features
- Live gold & silver prices in INR
- Auto-refreshes every 60 seconds
- Gold calculator by grams (22K / 24K)
- Price history with chart and table
- Fully responsive

## Tech Stack
- React
- React Router
- Recharts
- Custom Hooks
- LocalStorage

## Live API
Data from [gold-api.com](https://gold-api.com)

## Run Locally
npm install
npm run dev
